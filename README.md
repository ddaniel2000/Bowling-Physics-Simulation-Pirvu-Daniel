"START BY PRESSING R <br />
!! DISABLE CAPS_LOCK !! <br />
"----------------------" <br />
MOVE RIGHT - -> (RIGHT ARROW) <br />
MOVE LEFT - <-(LEFT ARROW) <br />
PUSH FORWARD - UP ARROW <br />
PUSH 1 TIME WITH FORCE - G <br />
JUMP - Space <br />
RESET - R <br />
GUTTER ON - C <br />
GUTTER OFF - X <br />

Good luck to hit the pins at the end of the lane";<br />
